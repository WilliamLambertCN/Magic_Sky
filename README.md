# Magic_Sky
Magic Sky Replace Software Pytorch Implemented. 

# requirement
    PYTHON 3.7  
    Pytorch 1.4  
    CUDATOOLKIT 10.1  
    CUDNN 7.6.5  
    PYQT 5.12  

The software can implement sky replace and auto-adapting color transfer on both image and video. Results will be saved in temp dir.
